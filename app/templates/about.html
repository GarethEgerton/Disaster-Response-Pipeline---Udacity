Summary


++

  • Pre-labelled tweets and text messages have been provided by Figure 8 who specialise in transforming text, image, audio, and video data into customized high-quality training data. This dataset was collated from communications across social media and messaging that took place following live disasters.

  • During and after a disaster there will typically be millions of tweets, text messages and other social media messages sent back and forth. Unfortunately it is difficult to distinguish between which of these are critical and require immediate attention from disaster response teams, and those that don't. Due to the sheer volume of messages, there is simply not enough time for human assesment of all communications when it is needed most. Typically only 1 in 1000 messages actually requries direct attention.

  • This project seeks to create a supervised machine learning model using natural language processing that is able to accurately categorise messages to be forwarded to relevant disaster response teams. It is a more sophisticated problem than that which a simple keyword search which would be able to solve, unable to capture necessary nuance in language.

  • The focus is initally on cleaning the raw data and constructing an ETL pipeline. A ML pipeline is constructed parameters fine tuned using gridsearch. This is the resulting Flask web app where predictions categorisations can be be made on new messages
